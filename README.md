## Features

* Modern glassmorphism design
* Smooth animated tab transitions
* Blur glass effect
* Clean dark UI
* Fully responsive layout
* Beautiful micro-interactions
* Pure Flutter implementation

## Tech Stack

* Flutter
* Dart
* AnimationController
* Custom UI Animations

```bash
flutter pub get
```
